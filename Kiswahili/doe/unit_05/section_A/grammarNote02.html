M - Mi
Singular Plural
mnada minada ‘auction(s)’

With this group of nouns, the singular is identified by the prefix m- and the plural is marked by mi-. Because there is another class whose singular is marked by m-, we need to distinguish the two. This class contains nouns that refer to body parts (mkono - mikono ‘arm/arms’), plant-life (mti - miti ‘tree/trees), and physical features like mountains (mlima - milima ‘mountain/mountains’), rocks (mwamba - miamba), rivers (mto - mito), etc….To distinguish it from class 1, we note that nouns in this class do not refer to living beings. They are predominantly plant life, physical features and some body parts. The plurals, class 4, are marked by the prefix mi.
